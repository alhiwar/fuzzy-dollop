الحوار الثقافي — موقع ثابت (React + Tailwind)
===========================================

هذا مشروع جاهز للنشر لموقع "الحوار الثقافي" (Static site) باستخدام React + Vite + Tailwind.
يحتوي على:
- صفحة رئيسية مع أقسام متعددة
- صفحة كاتب وصفحة مقال
- بحث داخلي وتبديل اللغة (ar/en)
- تذييل يحتوي على روابط تواصل افتراضية وبريد info@alhiwar.com

تشغيل محليًا:
1. تأكد أن لديك Node.js مثبتًا (نسخة 18+ أو 16+).
2. افتح مجلد المشروع في الطرفية:
   cd hiwar-website
3. ثبت الحزم:
   npm install
4. شغّل السيرفر التطويري:
   npm run dev
5. افتح المتصفح على العنوان الذي يطبع في الطرفية (عادة http://localhost:5173)

نشر مجاني (مقترح: Vercel):
1. أنشئ حسابًا في https://vercel.com
2. ارفع المستودع عبر GitHub أو قم بربط المجلد مباشرة من جهازك.
3. Vercel يتعرف على مشروع Vite ويجري النشر تلقائيًا.

تعديل روابط التواصل:
- افتح src/config.js وعدّل الروابط والبريد بسهولة.
