# الحوار الثقافي — Cultural Dialogue

موقع ثقافي عربي/إنجليزي يهتم بالفكر والأدب والبحوث والمقالات الثقافية.

## وصف المشروع

تم إنشاء الموقع ليكون مساحة هادئة للنشر الثقافي والفكري، مع التركيز على:

* الأدب
* الفكر والسياسة
* البحوث العامة
* البحوث الإسلامية
* المقالات الثقافية
* صفحات الكتّاب

الموقع يدعم اللغتين العربية والإنجليزية.
